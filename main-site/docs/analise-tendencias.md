# Análise de Tendências para Site Institucional (2026)

## Tendências aplicadas

1. Performance como prioridade de conversão
- Sites rápidos continuam sendo diferencial competitivo em aquisição.
- O projeto foi construído com Next.js estático, assets leves e animações sem bibliotecas pesadas.

2. Design orientado à credibilidade
- Layout com hierarquia clara, prova de valor no topo e linguagem objetiva.
- Estrutura focada em confiança: serviços, diferenciais, processo e chamada para contato.

3. Mobile-first real
- O acesso mobile já supera desktop no mundo, então o conteúdo foi planejado para leitura e ação em telas pequenas.
- Botões grandes, blocos curtos e navegação simples.

4. Conteúdo comercial direto
- Menos texto genérico e mais promessa concreta: o que entrega, como entrega, e qual suporte oferece.
- Fluxo pensado para reduzir dúvidas e acelerar decisão.

5. Movimento com propósito
- Animações de entrada e parallax suave para reforçar percepção moderna sem sacrificar velocidade.
- Respeito à preferência de redução de movimento do usuário.
